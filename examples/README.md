# Examples

## Purpose

This directory contains worked examples demonstrating the application of Project FAR.

Each example illustrates how the Foundational Architecture of Reasoning Analysis (FARA), the Foundational Analysis of Reasoning (FAR), and the Foundational Analysis of Reasoning Operations (FARO) can be applied to represent, investigate, compare, and analyze reasoning processes.

The examples are educational and illustrative.

They do not extend or modify the framework.

---

## Contents

- `simple-investigation.md` — Introduces the FAR methodology using a minimal example.
- `mathematical-proof.md` — Demonstrates deductive reasoning.
- `scientific-investigation.md` — Demonstrates empirical investigation.
- `historical-analysis.md` — Demonstrates evidential reasoning.
- `disagreement-analysis.md` — Demonstrates comparison and disagreement analysis using FARO.

---

## Reading Order

Readers new to Project FAR are encouraged to read the examples in the following order:

1. Simple Investigation
2. Mathematical Proof
3. Scientific Investigation
4. Historical Analysis
5. Disagreement Analysis

Each example introduces additional components of the framework while building upon the previous examples.
