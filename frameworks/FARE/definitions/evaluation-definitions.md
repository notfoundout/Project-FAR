# Evaluation Definitions

## Purpose

This document defines the canonical evaluation terminology used throughout the Formal Architecture of Reasoning Evaluation.

Unless explicitly stated otherwise, every evaluation-related term appearing elsewhere in FARE refers to the definitions given here.

---

# Definition 1 — Evaluation

An **evaluation** is the process of determining one or more assessments by applying specified evaluation criteria to one or more evaluation objects under specified evaluation conditions.

Evaluation is the fundamental process studied by FARE.

---

# Definition 2 — Evaluation Object

An **evaluation object** is any explicitly represented entity that may be evaluated.

Examples include:

- statements;
- hypotheses;
- arguments;
- models;
- evidence;
- observations;
- definitions;
- investigations;
- proofs.

The nature of an evaluation object does not affect its eligibility for evaluation.

---

# Definition 3 — Evaluation Criterion

An **evaluation criterion** is a formally specified standard used during evaluation.

Evaluation criteria determine how evaluation objects are assessed.

Different investigations may employ different evaluation criteria.

---

# Definition 4 — Evaluation Condition

An **evaluation condition** is a formally specified circumstance under which an evaluation is performed.

Evaluation conditions determine the context within which an assessment is valid.

Changing evaluation conditions may alter assessment outcomes.

---

# Definition 5 — Comparison

A **comparison** is the process of evaluating two or more evaluation objects relative to one another according to one or more evaluation criteria.

Comparisons may produce one or more assessments.

---

# Definition 6 — Assessment

An **assessment** is the explicit result produced by an evaluation.

Assessments represent evaluative conclusions rather than evaluation processes.

---

# Definition 7 — Evaluation Framework

An **evaluation framework** is a formally defined system specifying:

- evaluation objects;
- evaluation criteria;
- evaluation conditions;
- assessment procedures;
- assessment relationships.

FARE is an evaluation framework.

---

# Definition 8 — Evaluation Process

An **evaluation process** is the ordered sequence of operations performed during an evaluation.

Evaluation processes transform evaluation inputs into assessments.

---

# Definition 9 — Evaluation Context

An **evaluation context** is the complete collection of evaluation conditions, evaluation criteria, and evaluation objects relevant to a particular evaluation.

Every evaluation occurs within exactly one evaluation context.

---

# Definition 10 — Evaluation Result

An **evaluation result** is the complete collection of assessments produced by an evaluation.

An evaluation may produce:

- one assessment; or
- multiple assessments.

---

# Notes

These definitions establish the canonical evaluation vocabulary used throughout FARE.

Assessment theory, relationship theory, lifecycle theory, graph theory, and proofs shall reference these definitions unless explicitly stated otherwise.

Evaluation terminology defines the evaluation process itself.

Assessment terminology defines the products of that process.
