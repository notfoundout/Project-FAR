# FARE Meta-Theorems

## Purpose

This directory contains meta-theorems about the Formal Architecture of Reasoning Evaluation itself.

Regular FARE proofs establish results within the framework.

FARE meta-theorems establish properties of the framework, including dependency ordering, definition uniqueness, traceability, consistency, investigation soundness, and proof soundness.

# Classification Rule

A document belongs in this directory if its theorem is about FARE itself rather than about an assessment, relationship, graph, lifecycle, or evaluation object inside FARE.
