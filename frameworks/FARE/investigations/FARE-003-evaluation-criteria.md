# FARE Discovery Investigation

## Identifier

FARE-003

---

# Title

Evaluation Criteria

---

## Status

In Progress

---

# Purpose

Determine the nature of evaluation criteria.

The objective is to establish what criteria are and how they participate in evaluation.

---

# Motivation

Previous investigations established that evaluation compares an object against one or more criteria.

This investigation examines the criteria themselves.

---

# Central Question

What is an evaluation criterion?

---

# Candidate 1

Criterion as Rule.

Question

Is every criterion a rule?

Evaluation

Some criteria specify permissible behavior.

Others merely describe desirable properties.

Result

Partially Supported.

---

# Candidate 2

Criterion as Property.

Question

Is every criterion a property?

Evaluation

Many criteria concern properties.

Some concern relationships or behaviors.

Result

Rejected.

---

# Candidate 3

Criterion as Standard.

Question

Does every criterion establish a standard against which an object is compared?

Evaluation

Every investigated criterion establishes conditions for assessment.

Result

Supported.

---

# Candidate 4

Criterion as Constraint.

Question

Is every criterion a constraint?

Evaluation

Some criteria prohibit.

Others rank or measure.

Constraint appears too narrow.

Result

Rejected.

---

# Pattern Analysis

Every investigated criterion specifies one or more conditions against which an evaluation object is compared.

The criterion itself performs no evaluation.

It provides the standard enabling evaluation.

---

# Emerging Hypothesis

An evaluation criterion is an explicitly specified standard used to assess one or more evaluation objects.

---

# Candidate Taxonomy

Criteria may be:

- binary;
- ordinal;
- quantitative;
- qualitative;
- logical;
- operational;
- methodological;
- architectural.

---

# Observation

Evaluation criteria appear independent of evaluation objects.

The same criterion may be applied to multiple object classes.

Likewise, one object may be evaluated under multiple criteria.

---

# Consequences

If accepted:

Evaluation consists of:

- an evaluation object;
- one or more evaluation criteria;
- a comparison;
- an assessment.

This becomes the minimal evaluative architecture of FARE.

---

# Remaining Questions

Future investigations should determine:

- whether criteria possess formal structure;
- whether criteria admit dependency relationships;
- whether criteria themselves may be evaluated;
- whether criteria require justification.

---

# Current Status

The investigation remains active.

Current evidence supports treating evaluation criteria as explicit standards used during assessment.
