# FARE Discovery Investigation

## Identifier

FARE-004

---

# Title

Evaluation Relationships

---

## Status

In Progress

---

# Purpose

Determine the formal relationship between evaluation objects and evaluation criteria.

The objective is to establish the minimal structure required for evaluation.

---

# Motivation

Previous investigations identified:

- evaluation objects;
- evaluation criteria.

This investigation determines how these components interact during evaluation.

---

# Central Question

What relationship exists between an evaluation object and an evaluation criterion?

---

# Candidate Relationship 1

Membership.

Question

Does an object merely belong to a criterion?

Evaluation

Criteria are not sets of objects.

Evaluation requires assessment rather than membership.

Result

Rejected.

---

# Candidate Relationship 2

Satisfaction.

Question

Does evaluation determine whether an object satisfies a criterion?

Evaluation

Many evaluations determine satisfaction.

Others determine degree of satisfaction.

Result

Partially Supported.

---

# Candidate Relationship 3

Comparison.

Question

Does evaluation compare an object against a criterion?

Evaluation

Every investigated evaluation compares an object with an evaluative standard.

Comparison accommodates:

- binary outcomes;
- rankings;
- scores;
- qualitative assessments.

Result

Supported.

---

# Candidate Relationship 4

Transformation.

Question

Does evaluation transform the object?

Evaluation

Evaluation may produce an assessment while leaving the object unchanged.

Transformation is not required.

Result

Rejected.

---

# Pattern Analysis

Every investigated evaluation possesses:

- an evaluation object;
- one or more evaluation criteria;
- a comparison relation;
- an assessment.

The comparison establishes the relationship.

The assessment records the result.

---

# Emerging Model

Evaluation

├── Object

├── Criterion

├── Comparison

└── Assessment

The comparison relates the object to the criterion.

The assessment records the outcome of that comparison.

---

# Observation

One object may be compared against multiple criteria.

One criterion may be applied to multiple objects.

Evaluation therefore exhibits many-to-many relationships.

---

# Consequences

If accepted:

Evaluation becomes a formally decomposable process.

Every evaluation may be analyzed into:

- evaluated object;
- evaluation criterion;
- comparison relation;
- assessment.

Specialized evaluations become instances of this general structure.

---

# Remaining Questions

Future investigations should determine:

- whether multiple comparisons may contribute to a single assessment;
- whether assessments may themselves become evaluation objects;
- whether comparison possesses formal algebraic properties;
- whether evaluation relationships admit graph representations.

---

# Current Status

The investigation remains active.

Current evidence supports treating comparison as the formal relationship connecting evaluation objects and evaluation criteria.
