# FARE Discovery Investigation

## Identifier

FARE-002

---

# Title

Evaluation Objects

---

## Status

In Progress

---

# Purpose

Determine what kinds of entities may legitimately serve as the objects of evaluation.

The objective is to establish the domain over which evaluation operates.

---

# Motivation

The previous investigation proposed that evaluation consists of comparing an object against one or more criteria.

This investigation determines the nature of the object being evaluated.

---

# Central Question

What may be evaluated?

# Pattern Analysis

Every investigated object possesses identifiable properties capable of comparison against explicit criteria.

The nature of the object varies.

The evaluative process remains unchanged.

---

# Emerging Hypothesis

Evaluation is independent of the kind of object being evaluated.

Different object classes require different evaluation criteria.

The evaluation process itself remains invariant.

---

# Candidate Taxonomy

Evaluation Objects

- representations;
- relations;
- operations;
- reasoning states;
- reasoning processes;
- proofs;
- investigations;
- frameworks.

Additional object classes may be identified through future investigations.

---

# Consequences

If accepted:

FARE becomes a general theory of evaluation rather than a theory restricted to conclusions or proofs.

Evaluation procedures become reusable across every framework within Project FAR.

---

# Remaining Questions

Future investigations should determine:

- whether every evaluation object possesses a common formal structure;
- whether evaluation objects require explicit typing;
- whether object classes determine admissible evaluation criteria;
- whether evaluation objects admit hierarchical organization.

---

# Current Status

The investigation remains active.

Current evidence supports treating evaluation as object-independent while allowing evaluation criteria to vary according to object type.
