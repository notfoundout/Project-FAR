# FARE Discovery Investigation

## Identifier

FARE-012

---

# Title

Validity

---

## Status

In Progress

---

# Purpose

Determine the necessary and sufficient conditions under which an assessment possesses validity.

The objective is to establish validity as the first formal assessment dimension within FARE.

---

# Motivation

Previous investigations established the architecture of assessments.

This investigation begins the study of individual assessment dimensions.

Validity is investigated first because many other dimensions appear to presuppose or interact with it.

---

# Central Question

What makes an assessment valid?

---

# Candidate 1

Truth.

Question

Is validity equivalent to truth?

Evaluation

Truth concerns whether a representation corresponds to reality.

Validity concerns whether an assessment has been produced correctly according to its evaluation criteria.

An assessment may be valid even if later evidence changes beliefs about the evaluated object.

Result

Rejected.

---

# Candidate 2

Consensus.

Question

Does agreement determine validity?

Evaluation

Agreement neither guarantees nor is required for correct evaluation.

Consensus is independent of validity.

Result

Rejected.

---

# Candidate 3

Correct Procedure.

Question

Is an assessment valid whenever the evaluation procedure is correctly executed?

Evaluation

Correct procedure is necessary.

However, procedure alone does not determine which criteria were applied.

Result

Partially Supported.

---

# Candidate 4

Correct Application of Criteria.

Question

Is an assessment valid when the specified evaluation criteria are correctly applied to the evaluation object?

Evaluation

This definition applies uniformly across:

- representations;
- operations;
- proofs;
- investigations;
- frameworks.

Result

Supported.

---

# Pattern Analysis

Every investigated instance of validity requires:

- an identified evaluation object;
- explicitly specified evaluation criteria;
- correct application of those criteria.

Validity concerns the correctness of the assessment process relative to its own evaluative framework.

---

# Emerging Hypothesis

Validity is the property of an assessment resulting from the correct application of explicitly specified evaluation criteria to an evaluation object.

---

# Observation

Validity does not imply:

- truth;
- reliability;
- confidence;
- usefulness;
- completeness.

These remain independent assessment dimensions.

---

# Consequences

If accepted:

Validity becomes the foundational assessment dimension.

Subsequent investigations of:

- reliability;
- justification;
- admissibility;
- confidence;

may build upon it without conflating distinct evaluative properties.

---

# Remaining Questions

Future investigations should determine:

- whether validity is binary or graded;
- whether validity depends upon object type;
- whether validity admits formal proof;
- whether invalid assessments may nevertheless be useful.

---

# Current Status

The investigation remains active.

Current evidence supports treating validity as the property of an assessment resulting from the correct application of explicitly specified evaluation criteria.
