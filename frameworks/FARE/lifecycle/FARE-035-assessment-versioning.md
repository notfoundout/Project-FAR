# FARE Discovery Investigation

## Identifier

FARE-035

---

# Title

Assessment Versioning

---

## Status

In Progress

---

# Purpose

Determine the necessary and sufficient conditions under which an assessment version is created.

The objective is to establish the formal versioning model governing assessment evolution.

---

# Motivation

Previous investigations established:

- assessment revision;
- assessment refinement;
- assessment history.

Assessments evolve throughout investigations.

This investigation determines how successive versions are represented.

---

# Central Question

When does a new assessment version exist?

---

# Candidate 1

Every Modification.

Question

Does every modification create a new version?

Evaluation

Formatting and representational changes may not alter evaluative content.

Such changes do not appear to require new assessment versions.

Result

Rejected.

---

# Candidate 2

Every Revision.

Question

Does every revision create a new version?

Evaluation

Substantive revisions alter the assessment itself.

Current evidence supports this hypothesis.

Result

Supported.

---

# Candidate 3

Every Status Change.

Question

Does changing assessment status create a new version?

Evaluation

Status concerns lifecycle disposition.

The assessment itself remains unchanged.

Result

Rejected.

---

# Pattern Analysis

Assessment versions appear whenever substantive evaluative content changes.

Lifecycle events alone do not necessarily create new versions.

---

# Emerging Hypothesis

A new assessment version is created whenever the evaluative content of an assessment changes while preserving explicit historical continuity.

---

# Observation

Version identity differs from assessment identity.

Multiple versions may belong to the same evolving assessment.

Each version represents a distinct evaluative state.

---

# Consequences

If accepted:

Assessment evolution becomes explicitly versioned.

Historical reconstruction becomes unambiguous.

Refinement and revision become formally connected.

---

---

# Remaining Questions

Future investigations should determine:

- whether version identifiers require canonical representation;
- whether version graphs admit branching;
- whether versions preserve assessment equivalence;
- whether versions admit formal comparison.

---

# Current Status

The investigation remains active.

Current evidence supports treating versioning as the formal representation of successive substantive assessment states.
