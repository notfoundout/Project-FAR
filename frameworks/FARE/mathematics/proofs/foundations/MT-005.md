# Mathematical Theorem

## Identifier

MT-005

---

# Title

Zero Self-Distance

---

# Status

Accepted

---

# Objective

Show that every evaluation has zero distance from itself.

---

# Dependencies

- MDEF-004

---

# Statement

For every evaluation `E`, `d(E, E) = 0`.

---

# Proof

MDEF-004 states that the empty path has cost `0` and that an evaluation possesses zero distance from itself because the empty path and identity transformation have cost `0`.

Therefore `d(E, E) = 0` for every evaluation `E`.

---

# Corollaries

None.

---

# Consequences

Self-distance has a fixed base value.

---

# Notes

This does not imply that distinct evaluations cannot have zero distance.
