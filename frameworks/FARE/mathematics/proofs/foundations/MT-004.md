# Mathematical Theorem

## Identifier

MT-004

---

# Title

Every Geodesic Is an Evaluation Path

---

# Status

Accepted

---

# Objective

Show that geodesics have path structure.

---

# Dependencies

- MDEF-003
- MDEF-005

---

# Statement

Every evaluation geodesic is an evaluation path.

---

# Proof

MDEF-005 defines an evaluation geodesic as an evaluation path whose cost realizes the evaluation distance between its endpoints.

Therefore every evaluation geodesic is an evaluation path.

---

# Corollaries

None.

---

# Consequences

Path results may be applied to geodesics when their hypotheses are satisfied.

---

# Notes

No existence of geodesics is assumed.
