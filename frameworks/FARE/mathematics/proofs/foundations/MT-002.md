# Mathematical Theorem

## Identifier

MT-002

---

# Title

Reachability Is Transitive

---

# Status

Accepted

---

# Objective

Show that reachability composes through an intermediate evaluation.

---

# Dependencies

- MDEF-003

---

# Statement

If `E_2` is reachable from `E_1` and `E_3` is reachable from `E_2`, then `E_3` is reachable from `E_1`.

---

# Proof

Assume `E_2` is reachable from `E_1`.

By MDEF-003, there is an evaluation path `P_1` from `E_1` to `E_2`.

Assume `E_3` is reachable from `E_2`.

By MDEF-003, there is an evaluation path `P_2` from `E_2` to `E_3`.

By the path composition rule in MDEF-003, `P_2 ∘ P_1` is an evaluation path from `E_1` to `E_3`.

Therefore `E_3` is reachable from `E_1`.

---

# Corollaries

None.

---

# Consequences

Reachability chains may be shortened to endpoint reachability claims.

---

# Notes

This theorem does not establish symmetry.
