# Mathematical Theorem

## Identifier

MT-003

---

# Title

Path Composition Preserves Reachability

---

# Status

Accepted

---

# Objective

Show that compatible evaluation paths compose into a path between the outer endpoints.

---

# Dependencies

- MDEF-003

---

# Statement

If a path exists from `E_1` to `E_2` and a path exists from `E_2` to `E_3`, then a path exists from `E_1` to `E_3`.

---

# Proof

Let `P_1` be a path from `E_1` to `E_2`.

Let `P_2` be a path from `E_2` to `E_3`.

By MDEF-003, if `P_1` terminates at the evaluation where `P_2` begins, then `P_2 ∘ P_1` is an evaluation path.

Therefore an evaluation path exists from `E_1` to `E_3`.

---

# Corollaries

Reachability is transitive when expressed through compatible paths.

---

# Consequences

Long paths may be built from shorter compatible paths.

---

# Notes

This theorem does not state that composed paths are geodesics.
