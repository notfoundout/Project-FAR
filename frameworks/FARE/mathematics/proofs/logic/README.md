# FARE Logic Proofs

## Purpose

This directory contains mathematical proofs concerning logical and meta-mathematical properties of FARE.

---

# Scope

Logic proofs may concern:

- consistency;
- dependency structure;
- expressiveness;
- decidability;
- completeness;
- proof-system behavior;
- framework-level invariants.

---

# Notes

This directory is reserved for active reviewed theorem development.

Historical material belongs in `proofs/archive/legacy/`.
