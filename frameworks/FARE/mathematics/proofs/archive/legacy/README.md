# Legacy Proof Archive

## Purpose

This directory preserves historical or superseded proof documents for provenance.

Legacy proofs are not part of the active theorem library.

---

# Rules

Archived proofs shall not be cited as dependencies by active theorems.

Archived proofs may be restored only through formal review, renumbering if necessary, and inclusion in the theorem index.

Proofs shall not be deleted merely because their numbering conflicts with a newer theorem sequence.
