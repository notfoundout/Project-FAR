# FARE Mathematics Examples

## Purpose

This directory contains illustrative examples for the FARE mathematics layer.

Examples help explain definitions, theorems, and proof behavior.

---

# Rules

Examples never define mathematics.

Examples never prove mathematics.

Examples never serve as theorem dependencies.

Examples may motivate conjectures or clarify accepted definitions.

---

# Notes

If an example appears to establish a general result, that result must be rewritten as a theorem and proven separately.
