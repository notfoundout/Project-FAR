# CRE-002: Prospective Multi-Pressure Reasoning System

Status: Preregistered package under review; execution prohibited until package lock
Protocol: CRP-1.0 with CRE-002 prospective deterministic extension
Semantic authority: Vocabulary Semantics Baseline 1.0
Scenario: CRE-002-SCENARIO-1.0

## Purpose

CRE-002 is the first prospective experiment permitted to rely on Vocabulary Semantics Baseline 1.0. It tests whether the three official candidate vocabularies can license and support traceable encodings of a previously unseen scenario containing semantic and operational pressures not exercised together in CRE-001.

## Pressure dimensions

- nested conditions;
- bounded nondeterministic choice;
- interleaved concurrent processes;
- defeasible authorization with explicit priority;
- provenance-sensitive facts;
- higher-order rule modification;
- bounded historical constraints;
- explicit terminal and deadlock outcomes.

## Execution prohibition

No compiler, mapping, lowering, verifier result, evaluator output, or scientific conclusion may be committed as an official CRE-002 result until the package manifest is complete, checksums are locked, repository health passes, and `execution-lock.json` is changed through a separately reviewed PR.

## Nonclaims
