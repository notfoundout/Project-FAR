# Comparative Representation Evaluation

Status: Accepted methodology area

This directory is the canonical location for blinded, preregistered comparative representation evaluation. The authoritative protocol is `protocol-v1.0.md`.

External-validation artifacts outside this directory remain preliminary, historical, and exploratory evidence. They are not Version 1.0 comparative experiments and must not be used as sufficient support for comparative distinctiveness, reproducible structural constraint, local necessity, relative economy, minimality, or universality.

## Contents

- `protocol-v1.0.md` — the frozen Comparative Representation Protocol v1.0.
- `experiment-registry.json` — canonical registry for comparative representation experiments.
- `schemas/` — machine-readable instruments for Version 1.0 submissions and records.
- `experiments/CRE-001/` — preregistration for Comparative Representation Experiment 1.
