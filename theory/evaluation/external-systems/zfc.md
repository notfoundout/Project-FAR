# External System Evaluation — ZFC Set Theory

## Overview

ZFC is a first-order axiomatic foundation for set-theoretic mathematics. It is a major external test because it is highly expressive while still having a clear language, axioms, semantics, and proof theory.

## Investigation

Determine whether reasoning in ZFC can be represented using the five FAR primitives.

## FAR Mapping

| FAR Primitive | Mapping |
|---|---|
| Investigation | Determine whether a set-theoretic statement follows from the ZFC axioms. |
| Representation | Formulas, variables, set-membership expressions, axioms, definitions, lemmas, and theorems. |
| Representational Structure | Syntax trees, quantifier structure, dependency structure, and proof dependency relations. |
| Interpretation | Model-theoretic interpretation of the language of set theory. |
| Reasoning Calculus | First-order proof rules plus the ZFC axiom set. |

## Pressure Point

ZFC is large and foundational, but its reasoning components are explicit. The main pressure is scale, not primitive type.

## Classification

`fits FAR`

## Justification

ZFC maps cleanly to FAR because it is an axiomatized first-order theory. Its formulas are representations, its syntax and proof dependencies supply structure, its models supply interpretation, and its proof rules plus axioms supply reasoning calculus.

## Sixth Primitive Assessment

No sixth primitive is currently indicated.

## Confidence

Provisional.

## Remaining Questions

- Whether independence results should receive a separate derived concept for model variation.
- Whether large-scale mathematical practice built on ZFC introduces pragmatic reasoning layers beyond formal derivation.
