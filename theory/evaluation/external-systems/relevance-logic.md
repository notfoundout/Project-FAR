# External System Evaluation — Relevance Logic

## Overview

Relevance logic restricts implication so conclusions must be relevant to premises. It pressures FAR through relevance constraints on admissible inference.

## FAR Mapping

| FAR Primitive | Mapping |
|---|---|
| Investigation | Determine whether an inference preserves relevant premise-conclusion connection. |
| Representation | Formulas, premises, conclusions, implication claims, and relevance markers. |
| Representational Structure | Dependency and relevance relations between premises and conclusions. |
| Interpretation | Semantics for relevant implication. |
| Reasoning Calculus | Inference rules blocking irrelevant implication and unrestricted explosion. |

## Pressure Point

The pressure is not representation but admissibility: relevance must be encoded in the calculus.

## Classification

`conservative extension`

## Justification

Relevance logic requires a specialized admissibility policy, but that is a Reasoning Calculus extension supported by explicit dependency structure.

## Confidence

Provisional.
