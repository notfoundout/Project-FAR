# External System Evaluation — Scientific Hypothesis Testing

## Overview

Scientific hypothesis testing evaluates hypotheses against observations using experimental design, statistical tests, model comparison, and evidential thresholds.

## FAR Mapping

| FAR Primitive | Mapping |
|---|---|
| Investigation | Determine whether evidence supports, rejects, or fails to reject a hypothesis. |
| Representation | Hypotheses, observations, data, models, null hypotheses, alternatives, test statistics, and thresholds. |
| Representational Structure | Experimental design, variable relations, sampling structure, and model-data relations. |
| Interpretation | Statistical and methodological interpretation of evidence. |
| Reasoning Calculus | Test procedures, decision thresholds, model comparison, and evidential update rules. |

## Pressure Point

The pressure is evidential interpretation under uncertainty and methodological convention.

## Classification

`conservative extension`

## Justification

Scientific hypothesis testing requires statistical interpretation and evidential decision rules. These are conservative extensions of Interpretation and Reasoning Calculus.

## Confidence

Provisional.
