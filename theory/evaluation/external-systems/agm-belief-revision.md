# External System Evaluation — AGM Belief Revision

## Overview

AGM belief revision studies how a belief set should change when new information is accepted. It is a strong test for FAR because the target of reasoning is not only conclusion derivation, but rational transition between belief states.

## Investigation

Determine whether belief revision can be represented using the five FAR primitives.

## FAR Mapping

| FAR Primitive | Mapping |
|---|---|
| Investigation | Determine how a belief set should change after new information is introduced. |
| Representation | Belief sets, input propositions, revision operations, contraction operations, and resulting belief states. |
| Representational Structure | Inclusion, consistency, entrenchment, priority, and transition relations among beliefs. |
| Interpretation | Semantic or epistemic interpretation of belief states and accepted propositions. |
| Reasoning Calculus | AGM postulates, revision operators, contraction operators, and selection policies. |

## Pressure Point

The pressure is state revision rather than static inference. The system evaluates rational change in a represented belief state.

## Classification

`conservative extension`

## Justification

Belief states are representable, revision relations are structural, and AGM postulates function as calculus constraints. FAR needs a domain-specific state-transition calculus, but that is already within Reasoning Calculus and Representational Structure.

## Sixth Primitive Assessment

No sixth primitive is currently indicated.

## Confidence

Provisional.

## Remaining Questions

- Whether epistemic entrenchment should become a derived concept.
- Whether iterated belief revision requires additional transition-history machinery.
