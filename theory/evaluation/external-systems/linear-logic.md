# External System Evaluation — Linear Logic

## Overview

Linear logic treats assumptions as resources rather than freely reusable premises. It stresses FAR through resource-sensitive structure.

## FAR Mapping

| FAR Primitive | Mapping |
|---|---|
| Investigation | Determine whether a conclusion follows under resource-sensitive assumptions. |
| Representation | Formulas, resources, contexts, sequents, connectives, and proof steps. |
| Representational Structure | Resource accounting, context splitting, ordering, and dependency relations. |
| Interpretation | Resource-sensitive semantics such as phase spaces or categorical models. |
| Reasoning Calculus | Linear sequent rules controlling use, duplication, and disposal of assumptions. |

## Pressure Point

The pressure is resource accounting in the representational structure and calculus.

## Classification

`conservative extension`

## Justification

Linear constraints require specialized structural rules, but those rules belong to Reasoning Calculus and Representational Structure. No sixth primitive is indicated.

## Confidence

Provisional.
