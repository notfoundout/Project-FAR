# External System Evaluation — Type Theory

## Overview

Type theory evaluates reasoning through terms, types, judgments, contexts, and construction rules. It pressures FAR because propositions may be interpreted as types and proofs as terms.

## Investigation

Determine whether type-theoretic judgment and construction can be represented using the five FAR primitives.

## FAR Mapping

| FAR Primitive | Mapping |
|---|---|
| Investigation | Determine whether a judgment is derivable in a typing context. |
| Representation | Terms, types, contexts, judgments, constructors, eliminators, and proof terms. |
| Representational Structure | Context extension, dependency, term formation, typing relation, and substitution structure. |
| Interpretation | Type-theoretic semantics, including propositions-as-types where applicable. |
| Reasoning Calculus | Formation, introduction, elimination, computation, conversion, and substitution rules. |

## Pressure Point

The strongest pressure is that meaning and derivability are tightly connected. In type theory, interpretation and calculus interact more deeply than in ordinary first-order systems.

## Classification

`conservative extension`

## Justification

Type theory requires specialized handling of contexts, dependent judgments, construction semantics, and computation rules. These are domain-specific extensions to Representational Structure, Interpretation, and Reasoning Calculus. None requires a new primitive outside the five FAR categories.

## Sixth Primitive Assessment

No sixth primitive is currently indicated.

## Confidence

Provisional.

## Remaining Questions

- Whether definitional equality and computation should be registered as a derived concept.
- Whether dependent contexts require a more formal FAR schema.
