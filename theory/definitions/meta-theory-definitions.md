# Meta-Theory Definitions

## Purpose

This document establishes canonical meta-theoretic terminology used by Project FAR.

Only concepts whose meanings have been sufficiently investigated shall appear in this document.

All definitions contained herein are canonical for meta-theoretic usage.

---

# Definition 1 — Formal Theory

A **formal theory** is a formally specified system consisting of explicitly identified formal objects together with rules governing their construction, interpretation, and derivation.

---

# Definition 2 — Formal Object

A **formal object** is an explicitly identifiable object belonging to a formal theory.

Formal objects include, but are not limited to:

- concepts;
- statements;
- rules;
- proofs;
- derivations; and
- models.

---

# Definition 3 — Primitive Concept

A **primitive concept** is a concept whose meaning is not defined in terms of other concepts within the same formal theory.

Primitive status is always relative to a particular formal theory.

---

# Definition 4 — Defined Concept

A **defined concept** is a concept whose meaning is specified in terms of previously established concepts.

---

# Definition 5 — Statement

A **statement** is a formal object capable of possessing a truth value within a formal theory.

Statements include:

- axioms;
- propositions; and
- theorems.

---

# Definition 6 — Rule

A **rule** is a formal object specifying an admissible operation within a formal theory.

Inference rules are a particular class of rules.

---

# Definition 7 — Derivation

A **derivation** is the finite construction of a formal object from previously established formal objects according to the admissible rules of the theory.

---

# Definition 8 — Proof

A **proof** is a derivation establishing a specified statement.

Every proof is a derivation.

Not every derivation is necessarily a proof.
