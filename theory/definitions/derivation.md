# Definition — Derivation

## Purpose

This document defines derivation within the shared meta-theoretic layer.

---

# Definition

A **derivation** is an explicitly specified sequence of explicit steps beginning with previously accepted results and terminating in a target result.

A derivation itself establishes no claim of validity.

Validity is established only through proof.

---

## Consequences

A derivation:

- begins only from previously accepted results;
- explicitly identifies every intermediate step;
- terminates in a target result;
- does not itself establish correctness.

Whether a derivation is valid is determined only through successful proof.
