# Proofs

## Purpose

This directory contains the formal proofs of the theorems established within Project FAR.

Separating proofs from theorem statements allows theorem documents to remain concise while permitting proofs to develop independently as the formal theory grows.

---

# Organization

Each theorem should have its own proof document.

Recommended naming convention:

```text
proof-t-identifier.md
```

For example:

```text
proof-t-universal-architecture.md
proof-t-reasoning-independence.md
```

---

# Proof Structure

Each proof should contain:

- The theorem being proved.
- Dependencies.
- Assumptions.
- Formal proof.
- Notes (if necessary).

---

# Guiding Principles

Proofs should be:

- Complete.
- Explicit.
- Logically rigorous.
- Traceable to their dependencies.

Whenever possible, every inference should identify the definition, axiom, proposition, or theorem upon which it depends.
