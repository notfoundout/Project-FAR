# T-015 — Explicit Reasoning Meta-Theorem

## Status

Established for explicit reasoning systems satisfying Project FAR scope assumptions.

---

## Statement

Every explicit reasoning system satisfying Project FAR scope assumptions is representable as a FAR model.

---

## Proof

Let `S*` be an explicit reasoning system satisfying Project FAR scope assumptions.

By the scope assumptions, `S*` contains or implies:

1. an objective or question under which reasoning is conducted;
2. explicit reasoning objects;
3. relations among those objects;
4. interpretation of those objects;
5. rules or standards governing admissible transitions.

Map these to the FAR primitive architecture:

- objective or question maps to Investigation;
- explicit reasoning objects map to Representations;
- relations among objects map to Representational Structure;
- interpretation of objects maps to Interpretation;
- rules or standards map to Reasoning Calculus.

By the Representation Theorem, the reasoning process generated by these components admits a FAR representation.

By the FAR Model Theory definition, a structure satisfying the FAR axioms is a FAR model.

The mapped structure satisfies the FAR axioms because each required primitive component is supplied by the scope assumptions.

Therefore `S*` is representable as a FAR model.

---

## Corollary

FAR is calculus-neutral. Logical, probabilistic, empirical, legal, mathematical, and informal reasoning systems can be represented when they satisfy explicit scope conditions.

---

## Limitation

The theorem does not apply to systems that lack explicit objects, lack interpretable content, lack structural relations, or lack any rule or standard of transition.
