# Universal Reasoning Structure Test

Status: Provisional

## Purpose

This test harness evaluates the central Project FAR question: is there a universal structure of explicit reasoning?

Each fixture maps an external reasoning system into FAR primitives:

- Investigation
- Representation
- Representational Structure
- Interpretation
- Reasoning Calculus

## Required fixture observations

Each machine-readable example records:

- what maps cleanly;
- what requires assumptions;
- what is not yet represented;
- whether the system currently fits FAR, extends FAR, falsifies FAR, or remains a draft fixture.

## Current fixture set

The initial v0.2.0 fixture set covers classical logic, first-order logic, Bayesian reasoning, scientific reasoning, legal reasoning, abductive reasoning, analogical reasoning, and non-monotonic reasoning.

These fixtures are not proof that FAR is universal. They are reproducible test cases for comparing reasoning systems under the same primitive architecture.
