# Universal Reasoning Structure Test

Status: Provisional

This test framework evaluates whether explicit reasoning systems share a universal structure expressible in FAR.

Each reasoning-system fixture maps an external system into:

- Investigation: the objective or question being pursued;
- Representation: explicit items participating in reasoning;
- Representational Structure: relations among represented items;
- Interpretation: meaning assigned to representations;
- Reasoning Calculus: transition or inference standards.

A fixture records whether the system fits FAR, extends FAR, falsifies FAR, or remains a draft test case.
