# Applications

## Purpose

This directory records controlled applications of Project FAR to existing reasoning systems.

Applications are not independent proofs of Project FAR. They test whether existing reasoning systems can be represented using the FAR tuple:

```text
<I, Rep, S, Int, C, T>
```

where:

- `I` is investigation;
- `Rep` is representation set;
- `S` is representational structure;
- `Int` is interpretation;
- `C` is reasoning calculus;
- `T` is reasoning trace.

## Current Applications

- Classical logic
- Predicate logic
- Bayesian reasoning
- Scientific reasoning
- Legal reasoning
- Everyday argument
