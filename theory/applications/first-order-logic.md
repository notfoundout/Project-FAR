# Application — First-Order Logic

## FAR Mapping

A first-order logic argument can be represented as:

```text
<I, Rep, S, Int, C, T>
```

where:

- `I` is the objective of deriving or evaluating a quantified claim;
- `Rep` includes variables, quantifiers, terms, premises, and conclusions;
- `S` records formula formation, scope relations, dependencies, and inference order;
- `Int` assigns domains, denotations, extensions, and truth conditions;
- `C` contains the applicable first-order inference rules;
- `T` records instantiation, generalization, substitution, and inference steps.

## Result

First-order logic fits FAR because scope, domain assignment, interpretation, and inference rules can all be represented explicitly.

## Boundary

Different semantic models for first-order logic can be represented as different interpretations over structurally similar representations.
