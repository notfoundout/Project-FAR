# Documentation

## Purpose

This directory contains the public documentation for Project FAR.

The documents in this directory introduce the project, describe its objectives, explain its organizational structure, and identify the canonical locations of its major concepts.

The formal architecture, methodology, comparative framework, and mathematical theory are maintained elsewhere in the repository.

---

## Contents

### Project

- `overview.md` — Provides an overview of Project FAR.
- `mission.md` — Defines the mission of Project FAR.
- `philosophy.md` — Defines the principles guiding the project's development.
- `scope.md` — Defines the intended scope and limitations of the framework.
- `style-guide.md` — Defines repository documentation standards.
- `canonical-map.md` — Identifies the canonical location of major concepts.

---

## Reading Order

Readers unfamiliar with Project FAR are encouraged to read the documentation in the following order:

1. `overview.md`
2. `mission.md`
3. `philosophy.md`
4. `scope.md`
5. `canonical-map.md`

The remaining directories provide the formal framework:

- `fara/` — Foundational architecture.
- `far/` — Investigation methodology.
- `faro/` — Comparative analysis.
- `theory/` — Formal theory.
