# External Validation Terminology Review

Project FAR distinguishes:

- **External-system evaluation:** Project FAR authors or internal evaluators apply FAR to a system external to FAR.
- **Independent validation:** evaluators independent of framework development execute a preregistered protocol without access to prior mappings.
- **Replication:** a separate execution reproduces a prior result.
- **Comparative validation:** FAR is compared against preregistered alternatives.

EV-001 through EV-029 are preliminary external-system evaluations. They are not independent validation unless independent evaluator provenance, blinding, protocol version, prior-mapping access, and replication status establish that stronger classification.
