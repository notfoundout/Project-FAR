# External Validation: Psychology

## Case Study

The Stroop effect is a canonical experimental result in which incongruent color-word stimuli typically produce slower or less accurate color-naming responses than congruent stimuli.

## FAR Mapping

| FAR role | Scientific realization |
| --- | --- |
| Representation | stimulus condition, response time, error, participant, trial |
| Representational structure | trial sequence, condition assignment, participant grouping, data table |
| Interpretation | response latency and accuracy treated as measures of interference under the task design |
| Investigation | test whether incongruence changes performance |
| Reasoning calculus | experimental design, exclusion rules, statistical comparison, effect estimation |
| Operation | presentation, measurement, filtering, aggregation, comparison, inference |

## Valid Inference

If a properly controlled experiment finds systematically slower color naming in incongruent than congruent trials, the data support an interference effect under that task and sampled population.

FAR can represent the conditions, measurements, aggregation rules, comparison, and bounded conclusion.

Result: PASS.

## Invalid Inference

A Stroop difference is treated as proof of one unique cognitive mechanism or as a diagnosis of an individual disorder.

The behavioral effect is compatible with multiple explanatory models, and group-level task performance does not by itself establish an individual clinical diagnosis.

Result: PASS.

## Malformed or Scope-Violating Case

Trials are excluded after outcomes are inspected without a preregistered or independently justified rule, or reaction times from different tasks are pooled as though they measured the same variable.

FAR can expose changes in the calculus, representation, and investigation that undermine the original inference.

Result: PASS.

## Alternative Explanations and Limits

Observed differences may be influenced by reading ability, color vision, language, practice, fatigue, device timing, participant selection, or analytic choices. The effect alone does not uniquely identify its cause.

## Source Basis

The Stroop color-word task originates in experimental work on interference in serial verbal reactions and has been extensively replicated and modified. This report uses the core incongruent-versus-congruent comparison only.

## Final Outcome

CONDITIONAL PASS

Condition: the task design, operational definitions, exclusion rules, statistical method, and population scope must be explicit.
