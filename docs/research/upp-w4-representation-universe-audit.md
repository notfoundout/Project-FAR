# UPP-W4 admissible representation universe audit

## Decision

`E*` is frozen as the class of complete representation packages that are finitely described, effectively constructible, effectively usable, total over the registered query domain, stable in identity, and explicit about all required support machinery.

## Breadth

The universe is deliberately neutral about surface medium. It includes symbolic, graph, relational, functional, distributed, probabilistic, learned, and hybrid encodings. It permits different entity decompositions, temporal granularities, storage layouts, evaluation strategies, and parameterizations.

## Boundary

The quantified object is the complete package presented for faithful use, not only its visible data structure. Decoders, interpreters, schedulers, codebooks, query interfaces, stores, proof objects, and oracle claims must be declared. PR #286 will define and prove the machinery-closure operator; this workstream only freezes what qualifies for entry into that proof.

## Anti-circularity

No admissibility criterion mentions RCCD or any RCCD component. Membership in `E*` therefore does not establish kernel satisfaction. It establishes only that the representation is sufficiently specified and effective to be quantified over and audited.

## Three-valued discipline

A false criterion makes a candidate inadmissible. An unresolved criterion produces Unknown. Unknown is not promoted to admissibility merely because the encoding is sophisticated, learned, proprietary, or behaviorally successful.

## Registered attacks

The test suite attacks every criterion independently, rejects undeclared or unrealized oracle support, checks medium invariance, and scans operative criteria for kernel vocabulary. The checker also prevents later theorem claims from being imported into the result artifact.

## Nonclaims

This workstream does not prove machinery closure, representation equivalence, faithfulness, RCCD necessity, sufficiency, irreducibility, nontriviality, maximality, or the final universal theorem.
