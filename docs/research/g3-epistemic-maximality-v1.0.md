# G3 Epistemic Maximality v1.0

## Question

Does the requirement that relevant reasoning facts be accessible merely restrict Project FAR for convenience, or is some access condition logically required for any warranted structural classification?

## Result under validation

The G3 theorem proves an observation-level underdetermination result.

Suppose two source systems:

1. produce the same accessible observation;
2. have incompatible true structural classifications.

Any classifier restricted to those observations must return the same result for both systems. It therefore cannot be sound for both incompatible sources. This is independent of whether the proposed classification is RCCD, a competing kernel, or no kernel.

Consequently, when an incompatible observational twin has not been excluded, a determinate structural classification is not warranted. The correct adjudication is `Unknown`, not Pass and not Fail.

## What this establishes

The access boundary is not an arbitrary exemption added to protect FAR. Some discriminating access is a necessary condition for warranted determinate preservation and kernel judgments.

This yields a maximal epistemic result:

- G1 provides one kernel-checked composition of the relative semantic theorem.
- G2 quantifies the structural lower bound over every admissible, machinery-closed, fully faithful candidate representation, without finite vocabulary enumeration.
- G3 proves that systems lacking sufficient discriminating access cannot be used for a warranted determinate structural verdict by FAR or by a competitor.

The strongest supported universality claim is therefore:

> Across all reasoning systems for which the relevant structural facts are sufficiently accessible to support the independent full-faithfulness judgment, every admissible machinery-closed fully faithful representation realizes RCCD-equivalent obligations. Where those facts are not discriminatingly accessible, no determinate competing structural classification is warranted; the case remains Unknown.

## What this does not establish

The theorem does not reveal the metaphysical structure of inaccessible cognition. It does not prove that every hidden process is RCCD, that every physical process is reasoning, or that RCCD is the only possible ontology or implementation.

It establishes the strongest universality that can be warranted under explicit evidence-access conditions, while preserving inaccessible cases symmetrically for FAR and all competitors.

## Final claim taxonomy

1. **Unrestricted metaphysical universality:** not established.
2. **Open-world universality over admissible fully faithful representations:** established conditionally by G2.
3. **Maximal epistemically establishable universality:** established by combining G2 with the G3 underdetermination boundary.
4. **Inaccessible systems are RCCD:** not claimed.
5. **Inaccessible systems refute RCCD:** not claimed.
