## Purpose

Project FAR is an open research program investigating whether all structured, explicit, auditable reasoning can be represented by a common minimal architecture.

The documentation is versioned and updated as the research progresses.

Every major change to the theory is documented publicly.

## Current Status

The framework is under active development and should not yet be regarded as a completed formal theory.
