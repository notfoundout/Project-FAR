# Project FAR v0.3.1

## Highlights

Project FAR v0.3.1 is a maintenance release for the v0.3 series. It adds the repository command center, automation workflows, release-readiness reporting, dashboard metrics, repository indexing, and clearer health diagnostics while preserving the v0.3.0 theory and evaluation baseline.

## What Changed

- Made the root README the canonical command center.
- Added a generated README dashboard block.
- Added dashboard metrics and repository index generation.
- Improved generated report navigation and internal links.
- Improved repository health failure diagnostics.
- Added one-click GitHub Actions for repository maintenance workflows.
- Added release-readiness reporting.
- Added repository automation documentation.
- Added or confirmed canonical Make targets for dashboard generation and health checks.

## Theory

No theory changes.

This release adds manual GitHub Actions for:

- Repository Health;

These workflows are advisory and maintenance-oriented. They do not publish releases automatically and do not authorize theory changes.
