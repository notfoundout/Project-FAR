# Project FAR v0.3.0 Theory Freeze

Status: v0.3.0 freeze note.

Project FAR v0.3.0 freezes the current primitive-sufficiency evaluation baseline.

- The five primitives are not changed.
- Evidence registry, adversarial suite, expanded fixture analysis, independence analysis, and minimality analysis form the v0.3.0 baseline.
- Future v0.4.0 work should treat this as the internal validation baseline before external validation.
- Future changes must not silently rewrite v0.3.0 conclusions.

This freeze is not a claim that FAR is proven universal. It records the current internal evidence state: no analyzed case currently establishes a sixth primitive, one adversarial pressure remains unresolved, and future falsification remains possible.
