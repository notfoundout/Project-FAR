# Project FAR v0.2.0

Status: Draft

Project FAR v0.2.0 is the phase where FAR begins testing the universal-structure hypothesis: whether explicit reasoning systems share a universal structure expressible through Investigation, Representation, Representational Structure, Interpretation, and Reasoning Calculus.

The release focus is core reasoning infrastructure:

- FIR-style parsing for machine-readable FAR examples;
- structured proof-step statements and validation;
- stricter semantic proof-object checks with soft comparison where exact matching is not reliable;
- reasoning-engine dependency graphs, derivation trees, circularity detection, and machine-readable proof traces;
- fixture-based comparison of external reasoning systems;
- falsification criteria for disproving the current primitive architecture.
