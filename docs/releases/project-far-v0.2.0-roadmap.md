# Project FAR v0.2.0 Roadmap

Status: Provisional

v0.2.0 advances Project FAR from repository validation toward testable reasoning infrastructure.

Milestone targets:

- define FIR objects for representations, statements, proof steps, dependencies, transitions, derivations, reasoning-system mappings, and proof traces;
- document structural, rule-pattern, and semantic proof checking;
- parse minimal reasoning-system fixtures;
- emit machine-readable proof traces from the reasoning engine;
- define falsification criteria for the current primitive architecture.
