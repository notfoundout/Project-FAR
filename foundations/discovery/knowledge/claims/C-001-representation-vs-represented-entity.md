# C-001 — Representation Is Distinct from Represented Entity

## Status

Pilot Claim

## Claim

A representation is distinct from the entity, state, structure, or object it represents.

## Current Support Level

Supported by Phase II grounding work.

Not yet canonical.

## Origin

- GP-005 — State Grounding Investigation
- GP-008 — Representation Grounding Investigation

## Supporting Investigations

- GP-005 separated state from representation of state.
- GP-008 separated representation from represented entity.
- GP-009 separated representation from interpretation and meaning.
- GP-010 separated model from represented system.

## Supporting Audits

- AA-003 identified process/record and representation/entity-style category collapses in the Foundational Discovery Protocol.

## Counterexamples Considered

- different diagrams representing the same graph;
- different notations representing the same mathematical object;
- different encodings representing the same state;
- JSON key ordering changes that alter representation without altering represented state.

## Open Challenges

- Determine whether every representation requires interpretation.
- Determine whether representation is reducible to structure plus intended denotation.
