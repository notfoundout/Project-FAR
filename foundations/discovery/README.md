# Foundations Discovery

## Status

Research Index

## Purpose

This directory contains first-principles discovery work for Project FAR.

Its purpose is to ground the project below FAR, FARA, and FARO by asking what must exist before explicit reasoning is possible.

## Authority Boundary

This directory does not define canonical theory.

It does not canonize primitives.

It does not establish accepted necessities.

It records candidate discoveries, tests, reductions, counterexamples, proof attempts, and failures.

A result from this directory may be promoted only after satisfying the governing methodology in `../../methodology/research-doctrine.md` and `../../methodology/foundational-discovery-protocol.md`.

## Research Direction

Work in this directory proceeds from:

```text
weakest possible reasoning conditions
  ↓
necessity candidates
  ↓
reduction tests
  ↓
primitive candidates
  ↓
definitions
  ↓
lemmas
  ↓
theorems
  ↓
derived framework architecture
```

FAR, FARA, and FARO are not starting assumptions for this directory.

They are downstream structures to be tested, revised, derived, or rejected.

## Initial Research Question

The initial research question is:

> What is the weakest possible structure under which explicit reasoning can exist?

## Current Work Areas

- necessity candidates;
- primitive search;
- reduction attempts;
- minimal models;
- counterexamples;
- proof attempts;
- failed candidates;
- grounding links from accepted results to framework architecture.

## Non-Goals

This directory shall not:

- defend the current framework;
- assume FARA primitives;
- assume representation is primitive;
- assume state, transition, candidate, admissibility, or resolution are primitive;
- promote candidates without testing;
- hide failed attempts;
- convert useful design choices into necessities without proof.
