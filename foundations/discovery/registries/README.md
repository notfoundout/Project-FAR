# Discovery Registries

## Status

Research Index

## Purpose

This directory is reserved for future registry-backed discovery artifacts.

The registries are not implemented yet.

They are proposed as the eventual source of traceability for concepts, theorems, methodology rules, audits, investigations, and authority states.

## Intended Registries

Future registry areas may include:

- concepts;
- definitions;
- investigations;
- theorems;
- methodology rules;
- audits;
- framework components;
- authority decisions.

## Boundary

This directory currently records intended registry scope only.

It shall not be treated as populated metadata.

It shall not assign authority levels.

It shall not replace the current definition or theory files until a separate implementation decision is made.
