# Validation

## Purpose

This directory contains validation studies evaluating the applicability of Project FAR to existing reasoning frameworks.

Each study examines whether a reasoning framework can be represented using the Foundational Architecture of Reasoning Analysis (FARA), investigated using the Foundational Analysis of Reasoning (FAR), and analyzed using the Foundational Analysis of Reasoning Operations (FARO).

Validation seeks to evaluate the representational capabilities of Project FAR rather than the correctness of the frameworks being examined.

---

## Methodology

All validation studies follow the methodology defined in:

`template.md`

The template standardizes:

- Framework description
- Structural analysis
- Reasoning analysis
- Mapping to FARA
- Application of FAR
- Application of FARO
- Evaluation
- Validation summary
- Conclusion

Using a common methodology ensures that every reasoning framework is evaluated according to the same standards.

## Validation Objectives

Validation studies seek to determine:

- whether a framework can be represented by FARA,
- whether investigations can be conducted using FAR,
- whether reasoning processes can be analyzed using FARO,
- and whether the framework reveals opportunities for further research.

Validation does not modify Project FAR.

Potential improvements identified during validation should be documented separately and evaluated independently.
