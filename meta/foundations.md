# Foundations

## Purpose

This document defines the meta-theoretical foundations upon which Project FAR is constructed.

Unlike the theory documents, which describe reasoning, this document describes the assumptions, proof system, and logical framework used to develop and justify Project FAR itself.

Accordingly, every theorem, proposition, and proof appearing elsewhere in Project FAR is understood relative to the foundations established herein.

---

# Meta-Theory

Project FAR distinguishes between:

- the object theory; and
- the meta-theory.

The object theory consists of the formal concepts used to represent, analyze, and evaluate reasoning.

The meta-theory consists of the principles used to define, investigate, and justify the object theory.

Accordingly, statements concerning the validity of proofs, derivability, models, and logical consequence belong to the meta-theory rather than the object theory.

---

# Purpose of the Meta-Theory

The meta-theory exists to answer questions such as:

- What constitutes a valid proof?
- What is a model?
- What does it mean for one concept to be derivable from another?
- What constitutes logical equivalence?
- What assumptions are accepted without proof?
- What proof techniques are admissible?

The answers to these questions determine the standards by which the theoretical claims of Project FAR are evaluated.

---

# Foundational Principles

The development of Project FAR is governed by the following principles.

1. Definitions precede conclusions.

2. Every theoretical claim should be explicitly justified.

3. Primitive concepts should be minimized whenever possible.

4. Derived concepts should be preferred over additional primitives whenever equivalent expressive power is preserved.

5. Hidden assumptions should be identified and made explicit.

6. Competing explanations should be evaluated according to identical standards.

7. Methodological assumptions remain subject to investigation and revision.

---

# Components of the Meta-Theory

The meta-theory presently consists of the following components.

- Proof System
- Models
- Derivability
- Equivalence
- Dependency
- Research Methodology

Each component is defined in a separate document.

---

# Relationship to the Object Theory

The meta-theory does not describe reasoning.

Instead, it provides the framework within which claims about reasoning are investigated and justified.

The object theory depends upon the meta-theory.

The meta-theory does not depend upon the object theory.

---

# Current Status

The meta-theory remains under active development.

No component should be regarded as complete until its formal investigation has concluded.
